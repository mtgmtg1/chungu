# サポートファイル形式

Chunguは表抽出のために多様な入力形式をサポートしています。

## PDF

| 形式 | 備考 |
|--------|-------|
| `.pdf` | 単一または複数ページ。指定されたDPIで画像にレンダリングされます。 |

## 画像

| 形式 | 備考 |
|--------|-------|
| `.png` | スキャン文書に推奨 |
| `.jpg` / `.jpeg` | |
| `.gif` | 最初のフレームのみ処理 |
| `.bmp` | |
| `.webp` | |
| `.tiff` / `.tif` | |

## 音声

| 形式 | 備考 |
|--------|-------|
| `.mp3` | |
| `.wav` | |
| `.flac` | |
| `.aac` | |
| `.ogg` | |
| `.m4a` | |
| `.wma` | |

コスト: 音声**秒あたり1ポイント**。

## 動画

| 形式 | 備考 |
|--------|-------|
| `.mp4` | |
| `.avi` | |
| `.mov` | |
| `.mkv` | |
| `.flv` | |
| `.wmv` | |
| `.webm` | |
| `.m4v` | |

コスト: 動画**秒あたり3ポイント**。音声が抽出されて文字起こしされます。

## アーカイブ

複数ファイルを一度にアップロードするには圧縮します:

| 形式 | 備考 |
|--------|-------|
| `.zip` | 最も一般的 |
| `.rar` | |
| `.7z` | |
| `.tar` / `.gz` / `.tgz` / `.bz2` | |

アーカイブ内のすべてのサポート形式ファイルが抽出されて処理されます。

## 制限事項

- 最大ファイルサイズ: アップロードあたり**200 MB**（管理者設定可能）
- 最大ページ数: ジョブあたり**2,000ページ**（管理者設定可能）
- サポートされていない形式は`400 Bad Request`を返します
