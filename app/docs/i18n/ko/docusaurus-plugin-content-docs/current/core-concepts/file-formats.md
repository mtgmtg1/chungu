# 지원 파일 형식

Chungu는 표 추출을 위해 다양한 입력 형식을 지원합니다.

## PDF

| 형식 | 비고 |
|--------|-------|
| `.pdf` | 단일 또는 다중 페이지. 지정된 DPI로 이미지로 렌더링됩니다. |

## 이미지

| 형식 | 비고 |
|--------|-------|
| `.png` | 스캔 문서에 권장 |
| `.jpg` / `.jpeg` | |
| `.gif` | 첫 프레임만 처리 |
| `.bmp` | |
| `.webp` | |
| `.tiff` / `.tif` | |

## 오디오

| 형식 | 비고 |
|--------|-------|
| `.mp3` | |
| `.wav` | |
| `.flac` | |
| `.aac` | |
| `.ogg` | |
| `.m4a` | |
| `.wma` | |

비용: 오디오 **초당 1 포인트**.

## 비디오

| 형식 | 비고 |
|--------|-------|
| `.mp4` | |
| `.avi` | |
| `.mov` | |
| `.mkv` | |
| `.flv` | |
| `.wmv` | |
| `.webm` | |
| `.m4v` | |

비용: 비디오 **초당 3 포인트**. 오디오가 추출되어 전사됩니다.

## 압축 파일

여러 파일을 한 번에 업로드하려면 압축하세요:

| 형식 | 비고 |
|--------|-------|
| `.zip` | 가장 일반적 |
| `.rar` | |
| `.7z` | |
| `.tar` / `.gz` / `.tgz` / `.bz2` | |

압축 파일 내의 모든 지원 형식이 추출되어 처리됩니다.

## 제한 사항

- 최대 파일 크기: 업로드당 **200 MB** (관리자 설정 가능)
- 최대 페이지 수: 작업당 **2,000 페이지** (관리자 설정 가능)
- 지원되지 않는 형식은 `400 Bad Request` 반환
